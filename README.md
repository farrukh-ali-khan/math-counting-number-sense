### 1. Count Forwards

- **File:** game-1-count-forwards/count_forwards.py
- **Description:**
  - Prompts user for a starting number (1â€“100).
  - Prints the next 10 consecutive numbers.
- **Run:**

```bash

```

### 2. Skip Counting Stars

- **File:** game-2-skip-counting-stars/skip_counting_stars.py
- **Description:**
  - Prompts user to enter a number (2, 5, or 10).
  - Draws 5 stars using Turtle graphics.
  - Displays the multiples of the number exactly below each star.
- **Run:**

```bash
python game-2-skip-counting-stars/skip_counting_stars.py
```

### 3. Compare Numbers

- **File:** game-3-compare-numbers/compare_numbers.py
- **Description:**
  - Prompts user to enter two numbers (1â€“1000).
  - Prints which number is larger, or if both are equal.
- **Run:**

```bash
python game-3-compare-numbers/compare_numbers.py
```
